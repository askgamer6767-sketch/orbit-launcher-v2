# Roadmap

## v0.1 Broad Prototype

- Buildable multi-module launcher project.
- HOME launcher entry point.
- App index, launch flow, fuzzy search, command palette.
- Home grid, adaptive dock, theme studio, gesture studio, productivity, settings, privacy, widgets, backup, and plugin screens.
- Local heuristic prediction engine.
- Unit tests, UI smoke test, CI, and OSS docs.

## v0.2 Launcher Depth

- Real icon loading and adaptive icon pack support.
- Home page editing, drag/drop, folders, dock editing, desktop lock.
- Widget picker and hosted widget rendering.
- Gesture recognition on the home surface.
- Room migrations and more integration tests.

## v0.3 Productivity + Local AI

- Usage stats importer after explicit opt-in.
- Focus analytics and Pomodoro timer.
- Local model provider interface and optional TFLite sample.
- AI-generated categories from local metadata only.

## v0.4 Ecosystem

- Declarative plugin manifest validator.
- Theme pack import/export.
- Optional encrypted sync provider interface.
- Tablet, foldable, and large-screen layouts.
